JavaScript is Case Sensitive: the language keywords,variables, functions names, and other identifiers must always be typed with a consistent capitalization of letters.

The model for the browser display window is the window object

window object has three methods for creating dialog box:

Alert

Confirm

Prompt

Alert: open a dialog box which displays the parameter string and OK button

Confirm: opens a dialog box and displays the parameter and two buttons OK and Cancel

Prompt: open a dialog box and display its string parameter, along with a text box and two buttons, Ok and Cancel
